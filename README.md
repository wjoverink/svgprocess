# ProcessSvg

## Pre-requisite
- [Git](https://git-scm.com)
- [Node](https://nodejs.org)

### Set up frontend
1. Git clone this repo: `git clone`
1. Install dependencies: `npm install` 
1. Start frontend server: `npm start`
